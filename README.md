# Predicting Football Matches
### Overview
Predict whether a team will win or lose against another team by analyzing historical data from previous NFL games. 
### Key Components
_**WebScrape2.ipynb**_
  
    This notebook extracts raw stats from https://www.pro-football-reference.com/
 
_**nfl_df.csv**_
 
	Dataset generated from web scrape. From 2021-2024. Contains over 2,000 games. 
 
_**MLFootball.ipynb**_  
  
    It includes data preprocessing, feature engineering, model training (Random Forest), and evaluation steps. 

### Features

Data Collection: Automated web scraping of historical football match data.

Data Analysis: Exploratory data analysis (EDA) and visualization of trends and patterns.

Modeling: Building predictive models using techniques such as regression and classification.

Evaluation: Assessing model accuracy and performance using metrics like mean squared error (MSE) and accuracy score.
